Web de contacto "Bocaditos con Sazón"

Pagina web creada para facilitar la compra de comida rapidas a domicilio, para la comodidad de sus clientes al llevarle el pedido a la puerta de su casa. Deploy: Bocaditos con Sazón

ProgramandoAndo Website "Plataforma Educativa"

Aplicacion con fines educativos, proyecto realizado en equipo, fue creado con un equipo de 7 personas en un termino de 3 semanas, este proyecto permite tener una ruta de estudio para disintas profesiones concretamente creadas como escuelas que dentro tienen una serie de cursos ordenados para la comodidad del apendicizaje. Este proyecto se creo con las siguientes tecnologías para la Base de Datos usamos MongoDB/Mongoose, también para la construcción de CRUD se sacó Node/Express y en el front se trajo Redux/ReduxToolkit para el manejo de estado global, para el manejo de componentes y estado local se construyó con Reactjs y para el estilo CSS/Tailwind.
Deploy: ProgramandoAndo

Api Videogames RAWG

Aplicación con excelente UI y UX que muestra los detalles de tus videojuegos favoritos y permite crear tus propias descripciones de videojuegos mostrando los también al mismo lista. Los datos se obtuvieron con el consumo de Api RAWG para la creación de una API REST con su respectiva CRUD utilizando Express y los datos creados dentro de la app son guardados en una DB NoSQL MongoDB y todo el diseño y funcionalidad para el usuario fue realizada con Redux y React.
Deploy: Lista de Videogames
